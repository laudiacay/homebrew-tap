# Homebrew Tap for laudiacay

Custom Homebrew formulae by laudiacay.

## How to Use

```bash
brew tap laudiacay/tap
brew install <formula>
```

Or install directly:

```bash
brew install laudiacay/tap/<formula>
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| featurebox | Git worktree workflow manager for feature development |
